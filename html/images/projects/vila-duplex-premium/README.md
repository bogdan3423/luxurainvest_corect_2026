# Vila Duplex Premium - structura imagini

Pune imaginile pentru al doilea proiect in folderele de mai jos, dupa categoriile existente:

```text
exterior/
interior/
santier/
compartimentare/
```

Recomandare nume fisiere:

```text
exterior/vila-duplex-premium-exterior-01.webp
exterior/vila-duplex-premium-exterior-02.webp

interior/vila-duplex-premium-living-01.webp
interior/vila-duplex-premium-bucatarie-01.webp

santier/vila-duplex-premium-santier-01.webp
santier/vila-duplex-premium-santier-02.webp

compartimentare/vila-duplex-premium-compartimentare-parter.webp
compartimentare/vila-duplex-premium-compartimentare-etaj.webp
compartimentare/vila-duplex-premium-compartimentare-garaj.webp
```

Pastreaza numele cu litere mici, fara spatii, folosind cratima intre cuvinte.
