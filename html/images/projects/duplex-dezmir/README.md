# Duplex Dezmir - structura imagini

Pune pozele pentru acest imobil in folderele de mai jos:

```text
gallery/
  exterior/
  subsol/
  parter/
  etaj/
plans/
```

Recomandare nume fisiere:

```text
gallery/exterior/duplex-dezmir-exterior-01.webp
gallery/exterior/duplex-dezmir-exterior-02.webp

gallery/subsol/duplex-dezmir-subsol-01.webp
gallery/subsol/duplex-dezmir-subsol-02.webp

gallery/parter/duplex-dezmir-parter-01.webp
gallery/parter/duplex-dezmir-parter-02.webp

gallery/etaj/duplex-dezmir-etaj-01.webp
gallery/etaj/duplex-dezmir-etaj-02.webp

plans/duplex-dezmir-plan-parter.webp
plans/duplex-dezmir-plan-etaj.webp
plans/duplex-dezmir-plan-subsol.webp
```

Pastreaza numele cu litere mici, fara spatii, folosind cratima intre cuvinte.
